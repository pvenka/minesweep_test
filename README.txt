This is a simple reconstruction of the minesweeper game.
Three files mines.html, mines.js, and mines.css
The following options are there:

1) User can set number of mines and the grid size. Enter the values and click the 'reset' button.
2) It shows the score.  
3) Am limiting the number of rows or columns to 40 as anything larger may spill over the screen.
4) 'validate' button to check status of game
5) 'cheat' button to show location of mines.
